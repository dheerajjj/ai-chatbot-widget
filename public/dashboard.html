Widget Integration

Copy the code below and paste it just before the closing </body> tag on your website: 

<script src="http://localhost:3000/widget-fixed.js" data-api-key="YOUR_API_KEY_HERE" data-primary-color="#667eea" data-position="bottom-right"></script>
📝 Integration Example:
Here's how your code will look with your actual API key:

<script src="http://localhost:3000/widget-fixed.js" data-api-key="cb_abc123def456ghi789" data-primary-color="#667eea" data-position="bottom-right"></script>

💡 What each part does:
• data-api-key: Your unique API key (found in "API Configuration" section below)
• data-primary-color: Widget color theme (#667eea = purple)
• data-position: Where the widget appears (bottom-right corner)

🎨 Customize: You can change the color (e.g., #ff6b6b for red) or position (bottom-left, top-right, etc.)
